# Network Diagnostic Tool

A small Python command-line tool for basic network diagnostics.

## Features

This tool can:

- display hostname
- display local IP address
- test internet connectivity
- perform a simple DNS lookup test

## Technologies

- Python
- socket
- subprocess
- platform

## How to run

```bash
py src/network_check.py
```

## Purpose

This project was built as part of my learning journey into IT support and Python.
